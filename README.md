# ResQFood
A real-time Android application to reduce food waste and seamlessly connect food donors with local receivers.

## About the App
ResQFood (also known as Feed) is a mobile application developed to address the critical issue of community food waste.
It provides a secure platform where restaurants, households, and organizations can easily donate their surplus food instead of discarding it.
Local volunteers and receivers can browse live food listings in real time to claim donations.
By integrating real-time databases and interactive map locations, the app provides a scalable, digital solution to bridge the gap between 
food abundance and local hunger.

## Features
- Secure user signup & login authentication
- Dedicated Donor portal to easily submit food metadata and pickup details
- Live Receiver portal to browse available food listings in real time
- Google Maps API integration to display exact donor location coordinates
- Real-time cloud data synchronization and storage via Firebase
- User-friendly interface
